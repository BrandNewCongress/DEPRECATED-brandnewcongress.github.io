Let’s elect a Brand New Congress that works for the people.

We want to make sure there’s a Bernie Congress to go with a Bernie Presidency. Help us:

America needs an honest, functioning Congress to enact Bernie's program. But trying to win each congressional seat one-by-one is an impossible task. So let's run a unified congressional campaign that whips up the same enthusiasm, volunteerism and money as Bernie's presidential campaign and replace Congress all at once.

We'll build on all the tools, tactics and networks that staff and volunteers developed during Bernie's campaign. But to pull this off, we will need to give the volunteer movement more power and resources than any campaign ever before. This means volunteers will run offices, voter contact operations, and have access to all resources from the start. Get ready for the people's campaign.
